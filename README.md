# Modal dialogs

Dialogs components for freedomsex projects. Raw VUE components. Depend on global `bulma` and `vue-i18n` translator.

* ModalDialog
* AccentDialog
* InfoDialog
* ConfirmDialog

And some UI helpers.

## Nuxt|styleResources

```js
  modules: [
    '@nuxtjs/bulma',
  ],
  styleResources: {
    scss: ['./assets/styles/resource.scss'],
  },
  css: [
    { src: '@/assets/styles/bulma.scss', lang: 'sass' },
  ],
  styleResources: {
    scss: ['./assets/styles/resource.scss'],
  },

```
